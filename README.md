# Atlantic Wildlife Explorer

An interactive educational website exploring the rich marine biodiversity and ecosystems of the Atlantic Ocean. This project provides comprehensive information about marine animals, plants, mammals, and environmental insights about one of Earth's most important bodies of water.

## Overview

Atlantic Wildlife Explorer is a multi-page website designed to educate visitors about the Atlantic Ocean's diverse marine life and environmental significance. The site features engaging visuals, detailed species profiles, and up-to-date information about ocean conservation and environmental challenges.

## Features

- **Animals Section**: Explore diverse marine animal species including fish, invertebrates, and deep-sea creatures
- **Plants Section**: Learn about seaweeds, sea squirts, kelp forests, and other essential marine flora
- **Environment Section**: Understand the Atlantic's geography, climate influence, ecosystems, and conservation challenges
- **Mammals Section**: Discover marine mammals like dolphins, whales, seals, and manatees
- **Latest News**: Stay updated with current events and developments affecting the Atlantic Ocean
- **Interactive Design**: Hover effects, animations, and responsive layouts enhance user experience
- **Educational Content**: Comprehensive information about biodiversity, food chains, and conservation efforts

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- No server-side dependencies required

### Installation

1. Clone the repository:
```bash
git clone https://github.com/charlieijk/AtlanticWildlifeExplorer.git
```

2. Navigate to the project directory:
```bash
cd AtlanticWildlifeExplorer
```

3. Open `index.html` in your web browser:
```bash
open index.html
```
Or simply double-click the `index.html` file.
